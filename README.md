# CG-Algorithms-Easy-Implementation-in-C

> ### List Of Algorithms Included:

* 2D Transformations (Translation, Scaling, Rotation, Shearing, Reflection-w.r.t Origin, w.r.t X-axis, with Y-axis, w.r.t X=Y, w.r.t X=-Y)  
* Bezier Curves
* Boundary Filling
* Bresenham Line 
* Cohen Sutherland Line Clipping
* Cyrus Beck Line Clipping
* DDA Line
* Flood Filling
* Liang Barsky Line Clipping
* Midpoint Circle
* Midpoint Ellipse
* Scanline Polygon Filling
* Sutherland Hodgeman Polygon Filling
